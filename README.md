Course: Client-Side JavaScript

Description:
This project displays a toy car using HTML, CSS, and JavaScript.
A constructor function ToyCar was created with 10–15 properties describing the car.
The page shows the 1967 Ford Mustang GT and lets the user change its color interactively.

Files:
index.html – Webpage structure
style.css – Page and card styling
script.js – ToyCar constructor, describe method, and interaction
images/ – Mustang car image (mustang.jpg)

Features:
Constructor with multiple car properties
describe() method to display car details
Button to change car color
Responsive card layout
Step-by-step comments following class style

How to Use:
Open index.html in a browser.
View the Mustang GT details.
Click “Change Car Color” to update the car color on the page.
